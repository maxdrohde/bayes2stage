# Residual diagnostics for normal imputation model

Residual diagnostics for normal imputation model

## Usage

``` r
ppc_imputation_residuals(fit, data, imputation_covariates)
```

## Arguments

- fit:

  CmdStanMCMC fit object

- data:

  Data frame

- imputation_covariates:

  Character vector of imputation model covariates

## Value

List with diagnostic plots
